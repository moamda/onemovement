# Yii2 Instructions

- Follow Yii2 best practices
- Keep controllers thin
- Use service-based architecture
- Use ActiveRecord properly
- Use validation rules
- Use scenarios when needed
- Use transactions for critical operations
- Reuse existing helpers/services/components
- Follow existing module structure
- Use proper namespaces
- Use dependency injection when practical
- Use migrations for schema changes
- Avoid duplicated logic
- Keep implementations maintainable
